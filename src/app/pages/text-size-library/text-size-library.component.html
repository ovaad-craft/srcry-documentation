Text Size Library

The text size library consists of 7 sizes which each have 5 speeds. The 7 sizes are caption, widget, article, subtitle, title, headline and jumbotron. The syntax for using a text size goes as follows.

It's as simple as that, just use the word text with a dash after it followed by the size and speed. Lets look at some code to see how this works.

This library was specially designed to preserve a visual heirarchy between the different types of text respectively to the amount of space available on the screen. The demo on the home page already did a great job explaining how this works so there's no need to reitterate it. The following demonstration will allow you to select a text size and see how much growth occurs as the screen expands.

Just like the box sizes the text size is calculated with a static pixel value and a flexible value based on the overall screen size. Each text size has a core of 9px which is how they're closer in size on smaller devices because the flexible values equate to such a small size on smaller screens that there isn't as much of a difference between them. As the screen gets larger and the flexible values grows larger the difference between them becomes more evident.
