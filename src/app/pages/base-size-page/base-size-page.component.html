Base Size

The baseSize is a set of 6 properties that are used for defining the core size of your element. If for example you're taking a mobile first approach to your layout, this would be the size you want your element to be on mobile devices. The properties and their usage is defined in the chart below.

As mentioned on the overview page, there may be times when the size you're going for is in between two sizes which is why the nudge properties were created.
