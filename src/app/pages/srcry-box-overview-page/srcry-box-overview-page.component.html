srcryBox Class

The box size library can achieve a lot of things but sometimes a little more is required for your elements to scale as needed. The srcryBox class gives you an extra layer of functionality that allows you to fully control how your elements shrink and expand from one screen size to another. This is done through a set of properties defined on this class as CSS varialbles which hook into a series of equations resulting in the scaling behavior or your elements.

In order to use the srcryBox class you simply add it to your element's class attribute and define the srcryBox properties in your own class as shown in the example below.

Before we begin exploring the properties available on the srcryBox class it's worth pointing out that each property has 2 modifiers called nudge-chunk and nudge-slice. The purpose of including these modifiers is to slightly increase or decrease the size of your element should you run into a situation where one size is too small or adapts too subtly and the next size and or speed is too large or adapts too rapidly. A nudge-slice is a tiny flexible value that grows larger as the screen size expands and a nudge-chunk is equivalent to 8 nudge slices.
