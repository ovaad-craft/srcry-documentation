Box Size Library

The box size library is a collection of 198 size definitions that can be used to define the size of your elements, margins and padding. To make them more intuitive to use they're organized into nine sizes with two scales and eleven growth speeds. Before we go any further lets explain what each of these 3 parts are.

Size

The 9 sizes are called micro, xTiny, tiny, xSmall, small, loMed, hiMed, large and jumbo. Seeing that they're designed to be adaptable each name should be regarded arbitrarily as a relative range of sizes rather than exact pixel values.

Scale

As mentioned there are two scales for each size which is simply 1 or 2. The purpose is to broaden the range of each size by providing two core sizes to scale from.

Speed

The eleven speeds are used just like the scale settings which is applying any number between 1 and 11 to your settings. The speed controls how subtly or drastically an elements scales with the size of the screen with 1 being the slowest / most subtle and 11 being the fastest / most drastic.
