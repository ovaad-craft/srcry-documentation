Box Size Library

The box size library is a collection of 198 flexible settings which can be used to define the width and height of your elements as well as margins and padding. These settings are organized into nine sizes that each have two scales and eleven growth speeds to make using them intuitive and simple. Before we go any further lets explain what each of these parts are.

Size

The 9 sizes are called micro, xTiny, tiny, xSmall, small, loMed, hiMed, large and jumbo. Seeing that they're designed to be adaptable each name should be regarded as a relative range of sizes rather than exact pixel values.

Scale

The scale settings are used to broaden the range of each size by providing two core sizes to scale from. This is applied by simply adding a 1 or 2 to the size definition.

Speed

The speed setting controls how subtly or drastically the element will scale up and down with the size of the screen. This is applied by adding a number between 1 and 11 to the size definition.
