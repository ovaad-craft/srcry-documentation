Line Size Library

This library is designed to be used with your borders, outlines and SVG strokes. There are 11 sizes in total and unlike the box and text sizes there's no scale or speed settings. The syntax for using this library goes as follows.

The 11 sizes are fine, light, narrow, semiBold, bold, semiThick, thick, semiWide, wide, ultraWide and jumbo. Seeing that these sizes are fairly small in comparison to the box sizes they work well as drop shadow and padding values as well. Below is an example of how to use this library in your project.

One thing worth noting in regards to the next demonstration is partial pixel values are rounded to whole numbers when read with javascript. Due to this the readout value in the next demonstration will be slightly inaccurate but close enough to relate. Select the different line sizes and scale the screen to see how they adapt.

Like the box size and text size libraries this library has negative values as well which can be used in the same manner simply by adding a "-" to the end of the size definition.
