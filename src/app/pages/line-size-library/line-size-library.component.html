Line Size Library

This library is designed to be used with your borders, outlines and SVG strokes. There are 11 sizes in total and unlike the box and text sizes there's no scale or speed settings. The syntax for using this library goes as follows.

The 11 sizes are fine, light, narrow, semiBold, bold, semiThick, thick, semiWide, wide, ultraWide and jumbo. Seeing that these sizes are fairly small in comparison to the box sizes they work well as drop shadow values and padding as well. Below is an example of how to use this library in your project.

One thing worth noting here is partial pixel values are rounded to whole values when read with javascript. With that being said, the readout value in the following demonstration will be a little innacurate. Select the different line sizes to see how they scale across different screen sizes.

Like the box size and text size libraries this library has negative values as well which can be used in the same manner simply by adding a - to the end of the size definition.
