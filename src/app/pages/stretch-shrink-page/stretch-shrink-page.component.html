Stretch Shrink

The stretchShrink properties are used to make an element's dimensions shrink as the screen size expands. Similar to the squishGrowth properties, the stetchShrink-w properties make the width of an element shrink as the height of the page expands while the stretchShrink-h properties make the height of the element decrease as the width of the screen increases. This functionality is useful for handling situations where an element may start to grow out of control when scaling to larger screen sizes.
